# RAG Frontend

Dark hacker-themed React frontend for RAG Document Assistant.

## Setup

```bash
cd rag_frontend
npm install
npm run dev
```

Frontend runs on: http://localhost:3000
Backend required on: http://localhost:8000

## Structure

```
src/
  components/
    UploadBox.jsx
    SummaryPanel.jsx
    ChatBox.jsx
    Suggestions.jsx
  services/
    api.js
  App.jsx
  main.jsx
  styles.css
```
